# Chaatly 🌶️

Chaatly is a social platform that helps people discover, host, and join food-based meetups.

The goal is simple:

Meet people through food.

## Features

- Create meetups
- Browse meetups
- Join meetups
- Update meetup details
- Delete meetups

## Tech Stack

- FastAPI
- Python
- PostgreSQL (planned)
- React (planned)
